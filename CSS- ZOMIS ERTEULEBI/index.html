css-საზომი ერთეულები

css გააჩნია უამრავი საზომი ერთეული. განვიხილოთ ძირითადი მათგანი, ესენია:px,cm და mm,em,vh და vw, %-პროცენტი

px

css გააჩნია უამრავი საზომი ერთეული ერთ-ერთი ძირითადი საზომი ერთეულია PX(პიქსელი).
px წარმოადგენს ერთეულ წერტილს,მისი სიმრავლე ქმნის მონიტორზე სხვადასხვა გაოსახულებებს და ფორმებს.

cm და mm

სიგანე და სიმაღლე ასევე შეიძლება გამოისახოს სანთიმეტრებში(cm) და მილიმეტრებში(mm)

em

em-ეს საზომი ერთეული დაკაშირებულია შრიფტის ზომასთსან.
თითოეულ ბრაუზერს აქვს ნაგულისხმევი შრიფტის ზომა(16px),
ამიტომ 1em უდრის ნაგულისხმევ შრიფტის ზომას.
რამდენიც იქნება em იმდენჯერ გაიზრება შრიფტის ზომაც.

vh და vw

vh და vw ეს საზომი ერთეულები დაკავშირებულია viewport-თან.
viewport აღნიშნავს იმ სივრცეს რომელიც იხსნება ბრაუზერის ჩატვირთვისას.
მისი მთლიანი სიგანის და სიმაღლის ასათვისებლად საჭიროა გამოვიყენოთ vh და vw ერთეულები.
რამდენ ერთეულსაც გავუწერთ იმდენი პროცენტი სივრცე იქნება ათვისებული viewport-ში.

%-პროცენტი

%- ეს ერთეული ძირითადად დაკავშირებული არიან მშობელ ელემენტზე,
რამდენ პროცენტსაც მივანიჭებთ ელემენტს შესაბამის ადგილს დაიჭერს მშობელი ელემენტის სივრცეში.
